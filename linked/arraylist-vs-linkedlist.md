# ArrayList vs LinkedList

|                            | ArrayList | LinkedList |
| -------------------------- | --------- | ---------- |
| add() time complexity      | O(1)      | O(1)       |
| remove() time complexity   | O(n)      | O(1)       |
| get() time complexity      | O(1)      | O(n)       |
| contains() time complexity | O(1)      | O(n)       |
| 유리한 연산                     | 삽입, 조회    | 삽입, 삭제     |
